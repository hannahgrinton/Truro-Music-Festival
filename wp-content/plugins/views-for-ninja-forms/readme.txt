Show Ninja Forms Submissions in frontend using drag & drop View builder.

== Description ==

Ninja Forms Views lets you to show Ninja Forms submissions in frontend using drag & drop View builder.
You can create table view (list view coming soon).
It allows you add add content before & after the View.
You can set number of sumissions to show on page.

**Features:**
* Option to set form fields to show in View.
* Option to set custom label for table headings.
* Option to set number of submissions to show on page.
* Add pagination before or after the table.
* Show Pagination Info.

**How to use it?**

* Download and unzip the plugin
* Upload the entire ninja-forms-views/ directory to the /wp-content/plugins/ directory.
* Activate the plugin through the Plugins menu in WordPress.
* Create New Ninja Forms View from Views menu in admin sidebar.
* Save the View & copy the shortcode.
* Use the shortcode to show it on page.
